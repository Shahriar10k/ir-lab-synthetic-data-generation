PPG-DaLiA is a multimodal physiological dataset containing wearable sensor dataset
collected from subjects performing daily activites such as sitting, walking, cycling and working.
It was created to support researh on PPG-based heart rate estimation and human activity analysis
under realistic condition.

Why is PPG-DaLiA useful for synthetic activity data generation?
Because it contains multiple sensor data and subjects

The dataset is imbalanced because some activities, such as Transition and Lunch, contain
significant more samples than others like Stairs or Soccer. As a result the generative model may learn
the dominant activities better and generate more realistic syntheti samples for them, while struggling to acurately
model undertrepresented activities.

So Lunch having much more data than Stairs is not necessarily a data collection mistake.

It reflects reality.

# Should synthetic data reflect reality or balance activities?

It depends on the objective. If the goal is realistic simulation, the generated data should preserve the original activiy distribution.
If the goal is improving activity recognition, synthetic data can be used to augment underrepresented activities and
create a more balanced training set.

Preserve dataset: 
Advantage:

✅ Realistic population distribution

Disadvantage:

❌ HAR model still sees few stairs examples

Augment Dataset
Advantage:

✅ Better activity classifier training

Disadvantage:

❌ Less realistic distribution
